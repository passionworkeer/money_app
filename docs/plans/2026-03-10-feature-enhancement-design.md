# AI记账本 功能完善详细设计

## 项目概述
- **项目名称**: AI记账本
- **产品定位**: 智能语音记账应用，支持AI自动分类、预算管理、消费分析
- **目标用户**: 个人用户，日常消费记录管理者
- **当前版本**: v1.0.0

---

## Phase 1: 预算管理

### 1.1 核心功能

#### 1.1.1 预算设置
**用户故事**:
> 作为用户，我想设置每月预算，这样我可以控制支出并了解自己的消费习惯。

**功能描述**:
- 设置每月预算金额（数值输入，支持小数）
- 预算周期：每月1号自动重置
- 支持查看历史月份预算记录
- 首次使用时引导设置预算

**UI设计**:
```
设置页面新增"预算设置"区块:
┌─────────────────────────────────┐
│  💰 每月预算                    │
├─────────────────────────────────┤
│  预算金额                       │
│  ┌─────────────────────────┐   │
│  │ ¥ 5000.00              │   │
│  └─────────────────────────┘   │
│                                 │
│  预算周期：每月1号自动重置      │
│                                 │
│  [保存预算设置]                 │
└─────────────────────────────────┘
```

**数据模型**:
```dart
class Budget {
  final String id;
  final double amount;          // 预算金额
  final int month;              // 月份（1-12）
  final int year;               // 年份
  final DateTime createdAt;
  final DateTime updatedAt;
}
```

#### 1.1.2 预算进度显示
**功能描述**:
- 首页显示当月预算剩余额度
- 圆形或条形进度条展示使用比例
- 不同颜色区分健康状态

**UI设计**:
```
首页预算卡片:
┌─────────────────────────────────┐
│  💰 本月预算          3/31 ▼  │
├─────────────────────────────────┤
│  ¥ 2,150.00 / ¥ 5,000.00       │
│  ████████░░░░░░░░░░░░░░ 43%    │
│                                 │
│  ✓ 预算健康                     │
│  本月剩余 ¥ 2,850.00            │
└─────────────────────────────────┘
```

**颜色规则**:
| 使用比例 | 状态 | 颜色 |
|---------|------|------|
| 0-70%   | 健康 | #4CAF50 (绿色) |
| 70-90%  | 警告 | #FF9800 (橙色) |
| >90%    | 超支 | #F44336 (红色) |

#### 1.1.3 超支提醒
**功能描述**:
- 当月累计支出超过预算时显示警告
- 统计页面显示预算执行率
- 可选择开启/关闭提醒


## 技术债务

1. **状态管理**: 现有provider可复用，需扩展
2. **数据库**: 需新增budgets表
3. **路由**: 现有go_router配置无需修改
4. **测试**: 需补充新增功能的单元测试

---

## 设计原则

1. **渐进增强**: 核心功能优先，保持现有用户体验
2. **数据安全**: 导入导出需二次确认，防止误操作
3. **性能优先**: 搜索和筛选需考虑大数据量场景
4. **一致性**: 新增功能风格与现有UI保持一致
